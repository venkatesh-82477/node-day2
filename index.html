1.Explain CORS, Implement CORS.

A. Cross Origin Resource Sharing is a mechanism that supports secure requests and data transfers from outside origins(domain,scheme,port).In other words, CORS is a browser security feature that restricts cross-origin HTTP requests with other servers and specifies which domains access your resources.
Implementation :

        const cors = require('cors');
        app.use(cors({
         origin: '*'
        }));

7. What are JWT tokens?

A. JWT tokens:

  • JWT stands for JSON Web Token.
  • To acheive authentication in the API's, we use JWT tokens.
  • When any user logs-in by giving its username and password, then we will receive the username and password in the request body of the API.
  • Then using username and password we will create one token and assign it to the user.
  • Then whenever this user will be calling any API,then this token will be passed from front-end application,then from backend application we will verify this token is valid or not.
